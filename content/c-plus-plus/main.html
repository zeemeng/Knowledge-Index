About program security implications of accessing uninitialized pointers. Very clear and detailed explanation on differences between “lvalue” and “rvalue”.

An “lvalue” is a value which has an identifiable memory address associated with it. An “rvalue” is defined by exclusion to “lvalue”, that is any value that is not an “lvalue”. In programming languages such as C and C++, an “lvalue” can be implicitly converted to “rvalue”, while the reverse is not true.
